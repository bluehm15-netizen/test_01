# 🐍 Snake Game (HTML5 / JavaScript)

> **HTML, CSS, JS가 모두 포함된 단일 파일 스네이크 게임**  
> 별도의 라이브러리 없이 웹 브라우저만으로 실행할 수 있습니다.

---

## 📌 소개

이 프로젝트는 HTML5 Canvas 기반으로 제작된 심플한 스네이크 게임입니다.  
키보드 화살표 키를 이용해 뱀을 조작하며, 먹이를 먹으면 길이가 늘어나고 점수가 올라갑니다.  
벽 또는 자신의 몸과 충돌하면 게임이 종료됩니다.

---

## 🚀 특징

- **단일 HTML 파일 구조** (HTML + CSS + JS 포함)
- Canvas 기반 부드러운 애니메이션
- 점수 및 최고 점수(LocalStorage 저장)
- 게임 속도 자동 증가
- 일시정지 / 재시작 기능 포함
- 깔끔한 다크 테마 UI

---

## 🎮 조작 방법

|------|----|
